ECAD.jl Documentation

Access to blended daily public data from eca&d in Julia.

ECA&D (European Climate Assessment & Dataset, (Klein Tank et al., 2002)) is a project that provides information on changes in weather and climate extremes in Europe. In particular, it offers a blended daily dataset of multiple weather variables across thousands of stations.

Data policy

The dataset has a specific data policy which can be find here(pdf). To summarize, it must onl be used for non-commercial research and educational purposes, and scientific results based on these data must be submitted for publication for the open literature without delay.
