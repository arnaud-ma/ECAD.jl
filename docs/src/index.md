# ECAD.jl Documentation

*Access to blended daily public data from [eca&d](https://www.ecad.eu/) in Julia.*

[ECA&D](https://www.ecad.eu/) (European Climate Assessment & Dataset,  [KleinTank2002](@cite)) is a project that provides information on changes in weather and climate extremes in Europe. In particular, it offers a blended daily dataset of multiple weather variables across thousands of stations.


!!! warning "Data policy"
    The dataset has a specific data policy which can be find [here(pdf)](https://knmi-ecad-assets-prd.s3.amazonaws.com/documents/ECAD_datapolicy.pdf). To summarize, it must onl be used for non-commercial research and educational purposes, and scientific results based on these data must be submitted for publication for the open literature without delay.
